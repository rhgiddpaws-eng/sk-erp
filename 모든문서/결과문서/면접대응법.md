# SK ERP(또는 관련 기업) Web 연동 및 Web 분석 면접 대응 가이드

17년 차 이상의 탄탄한 시스템/게임 서버/풀스택 개발 경력과 최근의 '호텔 전산 리뉴얼', '유치원 ERP 기획/개발' 경험을 결합하면 **"ERP 도메인 전문가"는 아닐지라도 "ERP 현대화(Modernization) 및 대규모 데이터 연동/분석을 완벽하게 수행할 수 있는 아키텍트급 풀스택 실무자"**로 포지셔닝할 수 있습니다. 

풀스택을 원하고, "Web 연동 및 분석" 파트라면 **회계/구매 업무의 도메인 지식보다는, 레거시 ERP 시스템과 신규 Web 기반 시스템(또는 외부 시스템)을 어떻게 끊임없이 연동하고 방대한 데이터를 Web에서 어떻게 시각화/분석할 것인가**가 핵심입니다.

---

## 1. 면접 메인 컨셉 (Mindset & Positioning)

> **"나는 복잡한 회계 원리는 실무자에게 배우면 되지만, 그들이 원하는 대용량 트랜잭션 처리, 실시간 시스템 연동, 현대적인 Web 대시보드 구축은 이미 마스터한 개발자입니다."**

- **약점 방어:** 순수 ERP(SAP, 더존 등) 운영/개발 경력이 짧다는 점은 인정하되, "게임 서버에서의 대규모 동시접속/트랜잭션 처리 능력"과 "어떤 도메인이든 빠르게 분석하고 시스템화하는 설계 능력"으로 커버합니다.
- **최근 경험 어필:** 최근 주도했던 **'호텔 관리 전산 시스템 리뉴얼(PMS, 일종의 버티컬 ERP)'**과 **'유치원 ERP(Kindergarten-Transmission)'** 개발 경험을 적극적으로 내세워야 합니다.

---

## 2. 이력서 기반 "ERP" 맞춤형 스토리텔링

이력서 내의 경험들을 ERP 직무(Web 연동/분석)에 맞게 포장해야 합니다. 이 부분이 합격의 핵심입니다.

### ① 유치원 ERP (Kindergarten-Transmission) 
- **어필 포인트:** 최근 Next.js 16, React 19, TailwindCSS, Prisma 등을 활용하여 유치원 전문 ERP 웹을 직접 아키텍처링하고 개발했습니다.
- **연결 고리:** "최근 인사(HR), 예산/결산(Budget), 회계 전표(Transactions) 등의 ERP 핵심 모듈을 MSA 기반의 Web 프로젝트로 직접 설계하고 개발해 보며 ERP 도메인에 대한 이해도를 크게 높였습니다."

### ② 호텔 관리 전산 시스템 리뉴얼
- **어필 포인트:** 호텔 프론트부터 백오피스(재무, 객실, 영업)까지 이어지는 호텔 전산 시스템을 리뉴얼했습니다.
- **연결 고리:** "호텔 전산 시스템은 본질적으로 숙박업에 특화된 ERP(PMS)입니다. 기존의 낡은 시스템을 최신 Web 기술로 리뉴얼하며 레거시 연동과 데이터 마이그레이션을 주도적으로 수행 경험이 있습니다."

### ③ 엘리오소프트 (문서 변환 솔루션 및 설문 시스템)
- **어필 포인트:** Office, PDF, 한글 등 문서 실시간 변환 솔루션 개발. Kafka/Spring WebFlux 기반 대규모 설문 시스템.
- **연결 고리:** "ERP 시스템에서는 수많은 전자결재, 견적서, 세금계산서, 회계 리포트가 PDF나 엑셀로 변환되어야 합니다. 저는 이 부분에 특화된 기술을 보유하고 있습니다. 또한 Kafka를 활용한 비동기 데이터 파이프라인 경험은 ERP 'Web 연동/분석' 시 대용량 로그나 데이터를 연동하는 데 직결됩니다."

### ④ LG CNS 디지털트윈 (배터리 공장 시뮬레이션)
- **어필 포인트:** 공장 내부 설계 시뮬레이션.
- **연결 고리:** **(설비/장비 영역에 배정될 경우 초강력 무기)** "디지털 트윈 프로젝트를 통해 실제 공장의 설비 장비 데이터 파이프라인과 상태를 모니터링해 보았습니다. 설비 모듈에서 필요로 하는 데이터 연동과 Web 3D/2D 분석 뷰어 개발에 탁월한 역량을 발휘할 수 있습니다."

어느 부서일지 모르므로 각 부서의 일반적인 Web 연동/분석 니즈를 숙지하고 들어갑니다.

| 부서 후보          | Web 연동 및 분석 파트의 주요 업무 성격                                                                  | 면접 어필 전략                                                                                                                                                    |
| ------------------ | ------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **재무/관리 회계** | 타 시스템(인사, 구매, 영업)에서 발생한 전표 데이터를 실시간으로 집계 및 대시보드 화. 은행 연동(펌뱅킹). | "트랜잭션 일관성(ACID)과 무결성을 중시하는 게임 서버 및 결제 연동 경험이 깊습니다. 유치원 ERP에서도 현금출납장/전표(Transactions) 모듈을 설계했습니다."           |
| **구매 영역**      | 외부 협력사(Vendor) 포털 개발, 조달청/국세청 EDI API 연동, SCM 파이프라인 시각화                        | "Rest API, gRPC 등 외부 API 연동 경험이 풍부합니다. B2B 파트너들이 접근하기 쉬운 직관적인 Web 포털 및 분석 대시보드를 구축하겠습니다."                            |
| **장비/설비 영역** | 설비 IoT 센서 데이터 수집, 가동률 실시간 모니터링 통계/분석 대시보드                                    | "LG CNS 공장 디지털트윈 프로젝트 경험이 있어 설비 데이터에 대한 이해도가 높습니다. 센서 데이터를 Kafka/Redis 등으로 받아 Web 실시간 차트로 렌더링할 수 있습니다." |

## 4. 지원 분야: "Web 연동 및 Web 분석" 기술 면접 대비

### A. Web 연동 (API, MSA, Legacy Integration)
- **핵심 키워드:** RESTFul API, MSA 통신, Kafka, Redis, 인증/보안(SSO, JWT)
- **답변 전략:** "과거 C++, C#, ASP.NET부터 Node.js, Spring Boot까지 다양한 서버를 구축했습니다. 레거시(Legacy) ERP 서버가 C++이나 구형 Java로 되어 있더라도, 저는 이를 최신 Web 프론트(React/Next.js)와 완벽하게 연결하는 브릿지 API 및 MSA 구조를 만들 수 있습니다."

### B. Web 분석 (Data Visualization & Dashboard)
- **핵심 키워드:** 차트 시각화, 대용량 로그 처리, 실시간 통계.
- **답변 전략:** "대기업 납품용 대규모 설문 시스템을 개발하며 대규모 데이터를 집계해본 경험이 있습니다. 관리자 및 사용자가 직관적으로 Data를 볼 수 있게 UI/UX 프론트엔드 작업(React, Vue 등) 역량이 뛰어납니다."

### C. AWS 및 Kubernetes 활용 (현재 스터디 중 내용 방어)
- 솔직하되, 긍정적으로 발전 가능성을 제시합니다.
- **답변 전략:** "On-Premise 환경의 Linux/Windows 서버 구축 및 Docker 경험은 이미 풍부합니다. 최근 클라우드 환경 및 MSA의 표준이 된 Kubernetes(EKS)의 오토스케일링 및 컨테이너 오케스트레이션에 매력을 느껴 집중적으로 스터디하며 CI/CD 고도화 연습을 하고 있습니다. 현업에 투입되면 즉시 적응할 기본기는 갖췄습니다."

## 5. 최종 핵심 요약 스크립트 (1분 자기소개 & 포부 활용)

"안녕하세요. 17년 차 풀스택 개발자 OOO입니다.
저는 수많은 접속자가 오가는 대규모 모바일 게임 서버부터, 공공기관 메신저, 호텔 리뉴얼 관리 시스템까지 다양한 시스템의 아키텍처를 설계하고 Web/App 프론트엔드를 개발해왔습니다. 

최근에는 '유치원용 ERP(인사, 예산결산, 전표 모듈)'와 '호텔 프론트-백오피스 전산시스템'을 개발하며 ERP 도메인의 매력을 느꼈고, 복잡한 비즈니스 로직을 사용자가 가장 다루기 쉬운 Web 대시보드로 풀어내는 'Web 연동 및 분석'에 큰 자신감이 생겨 지원하게 되었습니다. 

특히, LG CNS의 공장 설비 디지털트윈 프로젝트 경험은 장비/설비 모듈에도, 엘리오소프트에서의 문서(PDF/Excel) 실시간 변환 엔진 개발 경험은 회계/구매 리포팅 파트에도 즉시 시너지를 낼 수 있다고 확신합니다. ERP 도메인의 깊은 지식은 훌륭한 동료들에게 열린 마음으로 배우되, 그 지식을 최신 Web 기술과 안정적인 클라우드/쿠버네티스 인프라 위에 튼튼하게 연동하고 구현해 내는 것은 제가 책임지겠습니다."
