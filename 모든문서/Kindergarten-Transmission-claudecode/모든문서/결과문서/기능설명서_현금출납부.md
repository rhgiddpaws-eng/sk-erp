# 기능 설명서 - 현금출납부 (Slip Management)

이 문서는 발표 및 영업 시연을 위해 작성된 가이드 문서입니다. 아이큐브 매뉴얼의 '현금출납부' 기능을 바탕으로 화면이 구성되었습니다.

## 화면 위치
- **메뉴 경로**: 좌측 사이드바 -> `Slip Management` (또는 메인 대시보드에서 접근)
- **URL**: `http://localhost:3000/dashboard/transactions`

## 핵심 기능 및 시연 포인트

1. **엑셀과 유사한 수기 입력 UI (빠른 입력 처리)**
   - 화면 하단에 엑셀처럼 곧바로 데이터를 칠 수 있는 **수기 입력란**(노란색 배경)이 있습니다.
   - **시연 방법**: 일자, 수입/지출, 계정과목, 적요, 금액을 마우스나 키보드(Tab)로 입력한 뒤 우측 끝의 `[+ 수기추가]` 버튼을 누릅니다.
   - **효과**: 페이지 전체가 새로고침 되지 않고 데이터가 바로 위의 리스트에 추가되며, 화면 맨 아래의 **수입 합계 / 지출 합계 금액이 실시간으로 증가**하는 모습을 보여줄 수 있습니다.

2. **조건 검색 및 필터링**
   - 상단의 검색창에서는 '유치원 선택', '기간 검색', '검색어(적요, 계정)'를 입력하여 필터링할 수 있습니다. 

3. **은행 계좌 연동 모방 및 핵심 편의 기능 (버튼들)**
   - 아이큐브 매뉴얼에 명시된 주요 버튼들을 우측 상단에 배치해두었습니다.
   - **계좌 가져오기**: 실제 은행에 연결해서 내역을 긁어오는 기능입니다. (발표 시 "은행 계좌를 연동하여 한 번에 수입/지출 내역을 불러온다"고 설명)
   - **전월 가져오기**: 매월 반복되는 고정비(임대료, 교직원 급여 등)를 지난달 내역에서 복사해옵니다.
   - **다중 분개**: 하나의 입금액을 여러 계정으로 쪼개서 분개하는 기능입니다 (예: 100만원 입금 시 수익자부담금 50만, 특성화비 50만 분리).
   - **입력마감**: 더 이상 전표를 수정할 수 없도록 마감 처리하여 회계 오류를 방지합니다.

4. **검증 및 에듀파인 전송 상태 파악**
   - 각 거래내역 리스트의 맨 우측에는 **에듀파인 전송 상태** (미전송 / 전송완료 / 에러)가 뱃지로 표시되어, 담당자가 쉽게 누락된 사항을 파악할 수 있도록 해줍니다.
   - 화면 맨 아래 **'계좌 조회 잔액과 일치합니까?' 검증 버튼**은 입력된 현금출납부의 기록과 실제 은행의 통장 잔고가 일치하는지 맞춰보는 용도로 사용됩니다.
