# 1. 이 프로그램은 도대체 왜 필요한가요? (프로그램의 목적)

유치원 선생님들은 매일매일 유치원 통장에 돈이 들어오고 나가는 걸 꼼꼼히 장부(현금출납부)에 적어야 해요. 그런데 이 장부를 유치원에만 적어두는 게 아니라, 나라에서 관리하는 **'에듀파인'**이라는 아주 어렵고 복잡한 시스템에 또 똑같이 타이핑해서 보고해야 한답니다. 게다가 은행 통장에 학부모님이 돈을 입금했는지, 전기세가 빠져나갔는지는 은행 사이트에 따로 들어가서 매번 확인해야 하죠.

선생님들은 아이들을 돌보는 것만으로도 너무 바쁜데, 이렇게 은행 확인 ➡️ 장부 작성 ➡️ 나라(에듀파인)에 보고하는 3단계를 매일 하는 건 너무 힘들어요.
그래서 이 프로그램(에듀파인 자동화 iCube 모방 시스템)은 **"은행 통장 내역을 자동으로 베껴오고 ➡️ 버튼 한 번 누르면 나라(에듀파인)에 알아서 똑같이 타이핑해주는 마법의 자동 비서"** 랍니다!

---

# 2. 어떤 원리로 작동하고, 어떻게 실행하나요?

- **은행 연결(스크래핑):** 선생님이 은행 아이디와 비밀번호를 한 번만 알려주면, 프로그램이 몰래 은행에 들어가서 통장 내역을 싹 베껴옵니다. (이걸 '스크래핑'이라고 불러요!)
- **장부 작성과 이름표 달기:** 베껴온 통장 내역에 "이건 간식비", "이건 교구 샀구나" 하고 이름표(계정과목)를 달아줍니다.
- **에듀파인 전송(RPA 로봇):** 이름표 달기가 끝나면, 눈에 보이지 않는 로봇(RPA)이 선생님을 대신해서 에듀파인 사이트에 자동으로 로그인하고 이 장부 내용을 똑같이 타이핑해서 등록해 줍니다. 선생님은 "전송 성공!" 글자만 확인하면 끝!

# 3. 매뉴얼의 버튼들은 무슨 역할을 하고, 어떻게 구현되나요?

우리가 똑같이 따라 만들 **유치원 아이큐브 매뉴얼**에 있는 주요 기능들을 유치원생도 알기 쉽게 설명해 드릴게요. 이것들이 어떻게 구현되는지도 같이 알려드릴게요!

### 🎈 [현금출납부 기능] (우리의 진짜 장부)
- **현금출납부 검색 기능**
  - 뭘 하는 건가요? 언제 어디에 돈을 썼는지 일기장 찾듯이 날짜나 이름으로 검색하는 버튼이에요.
  - 어떻게 구현했나요? `Next.js` 화면에 검색창을 만들고, 우리가 만든 데이터베이스(`PostgreSQL`)에서 원하는 날짜의 데이터만 쏙 뽑아오게 만들었어요.
- **전월자료 가져오기 기능**
  - 뭘 하는 건가요? 매달 똑같이 나가는 돈(월급, 전기세, 정수기 대여료 등)을 저번 달 장부에서 그대로 복사해오는 '복붙(복사 붙여넣기)' 마법 버튼이에요.
  - 어떻게 구현했나요? 저번 달 데이터를 DB에서 그대로 복사(Copy)해서 날짜만 이번 달로 바꿔서 저장하는 버튼을 만들었어요.

### 🏦 [회계 기능] (은행과 대화하는 곳) - ⚠️ 은행 연동 필요!
- **계좌연결 및 현황보기 / 계좌가져오기**
  - 뭘 하는 건가요? 우리 유치원 통장을 프로그램에 연결하고, **진짜 은행 통장에 찍힌 내역을 그대로 가져오는 버튼**이에요. 이게 없으면 선생님이 또 은행 사이트에 들어가야 하니까 꼭 필요해요! (은행 연동의 핵심!)
  - 어떻게 구현했나요? 백엔드 API (`/api/bank-scraping`)를 만들어서 선생님이 넣은 은행 로그인 정보로 빠른계좌조회 서비스를 '스크래핑'해와서 화면에 뿌려주도록 구현해요.
- **수입 다중분개**
  - 뭘 하는 건가요? 부모님들이 한 번에 보낸 원비 30만 원을 "원비 10만 원, 우유값 5만 원, 특별활동비 15만 원"으로 쉽게 쪼개서 기록하는 가위 같은 버튼이에요.
  - 어떻게 구현했나요? 화면(UI)에서 금액을 여러 칸으로 나눌 수 있는 팝업창을 만들고, 합계가 딱 맞으면 DB에 나눠서(Split) 저장하도록 만들었어요.
- **과오납 적용/해제**
  - 뭘 하는 건가요? 실수로 예산을 잘못 쓰거나 환불해 줄 때 쓰는 '되돌리기' 버튼이에요. 돈이 마이너스(-)로 계산되게 해줘요.

### 🤖 [부가 기능] (에듀파인과 대화하는 곳) - ⚠️ 에듀파인 연동 필요!
- **K-에듀파인 자동 전송**
  - 뭘 하는 건가요? 다 쓴 장부를 로봇 시켜서 **나라(에듀파인)에 자동으로 보내는(타이핑해주는) 로봇 출동** 버튼이에요! 이 프로그램의 가장 중요한 목적입니다!
  - 어떻게 구현했나요? `Playwright`라는 자동화 테스트 도구를 'RPA 로봇'처럼 썼어요. "에듀파인 전송" 버튼을 누르면 이 로봇이 에듀파인에 로그인하고 ➡️ 마우스를 클릭하고 ➡️ 결의서를 타이핑하고 ➡️ 저장까지 알아서 다 해줍니다!

### ⚙️ [설정 & 인사관리]
- **현금출납부 기초자료 설정 & 키워드 설정**
  - 뭘 하는 건가요? "이마트"라고 쓰여있으면 무조건 "급식비"로 이름표를 달아달라고 컴퓨터와 미리 약속을 정해두는 곳이에요. 나중에 장부 쓰기가 100배는 편해져요.
- **4대보험 고지내역 가져오기**
  - 뭘 하는 건가요? 선생님들 건강보험료나 국민연금이 얼마 나왔는지 매달 공단 홈페이지에서 알아서 가져와서 월급 계산할 때 쓰는 기능이에요. (이곳도 EDI 스크래핑 연동이 필요해요!)
